# Security Audit & Controls Assessment

A mock internal security audit evaluating an organization's controls against
compliance requirements, identifying gaps, and recommending remediations.

## Skills Demonstrated
- Conducting a controls and compliance assessment
- Applying security control categories (administrative, technical, physical)
- Risk identification and prioritization
- Writing clear, actionable remediation recommendations

## Summary
I reviewed the organization's scope, goals, and risk profile, worked through a
controls and compliance checklist, and documented which controls were in place,
which were missing, and what to prioritize.
